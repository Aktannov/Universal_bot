## Universal Bot


## Перед работой переименуйте .env.example на .env

### Установка зависимостей проекта, через poetry
```
poetry update
poetry install
poetry shell
```

### Установка зависимостей через venv
```
python3 -m venv venv
source venv/bin/activate
```

### Запуск проекта
```
python bot.py
```
